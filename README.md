# Traveloop

AI-powered collaborative travel planner built with Odoo.

## Features

- Multi-city itinerary planning
- Drag-and-drop trip stops
- Budget tracking
- Expense management
- Public trip sharing
- AI itinerary suggestions

## Tech Stack

- Odoo
- Python
- PostgreSQL
- XML/QWeb
- OpenAI API

## Installation

```bash
git clone https://github.com/USERNAME/traveloop-odoo.git
```

## Run

```bash
docker-compose up
```
